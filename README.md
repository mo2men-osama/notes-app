# Notes App

Simple notes application built with vanilla JavaScript, bundled with Webpack.

Live demo: https://moamen-notes-app.netlify.app

Getting started

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev-server
```

3. Build production bundle:

```bash
npm run build
```

Notes
- Data is stored in browser `localStorage`.
- Uses `uuid` and `moment` for IDs and date formatting.
